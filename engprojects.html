Engineering Projects

A sample of projects I have recently worked on

Airbag Detonation System

Project Summary

A system to detonate old automobile airbags by heating the airbags to activation temperature which triggers airbag propellant. The system came complete with a motion triggered flap gate, cooling tank and discharge conveyor.

Location

Near Lansing, MI, USA

Dry Carbon Scrubber Project

Project Summary

A series of columns containing activated carbon to scrub off-gases from an industrial plant so that the air could be discharged in an environmentally sustainable way. The project also included an exhaust fan system, emergency fire suppression system, and a set of carbon monoxide detectors.

Location

Veladero Mine site, Argentina

Continuous Mercury Retort

Project Summary

A system designed to continuously remove, distill and store mercury from processed mining precipitate. The project was designed to replace a series of retorts that operated in batches with a system that continuously fed and discharged material without needing to pause. The project also included a set of dry carbon scrubbers, mercury storage tanks, automatic feedbin and a water-cooled discharge screw.

Location

Veladero Mine site, Argentina

Wood Chip Drying Ovens

Project Summary

A set of two ovens to be used for drying wood chip samples for testing. The oven was redesigned to be readily accessible for a wheelchair-bound operator.

Location

Prince George, BC, Canada
